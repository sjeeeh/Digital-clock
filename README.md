# 🕒 Digital Clock

A simple and elegant desktop digital clock application built with **Python** and **PyQt5**. The clock updates every second and uses a custom seven-segment digital font for a realistic LED clock appearance.

## ✨ Features

- Real-time digital clock
- Updates every second
- Custom digital (7-segment) font
- Clean and minimal user interface
- Dark theme with bright LED-style text
- Built using PyQt5

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Digital_Clock.git
```

### 2. Navigate to the project folder

```bash
cd Digital_Clock
```

### 3. Install dependencies

```bash
pip install PyQt5
```

Or install using:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python DigitalClock.py
```
